Command Line Reference

The python -m icakad entry point provides a friendly CLI for non-developers.

Global Flags

  • --config <path> — load settings from a JSON file.
  • --token <value> — override bearer token at runtime.
  • --shorturl-base / --paste-base — switch between prod (linkove.icu) and staging (xp97.icu).

Short URL Commands

python -m icakad shorturl add slug https://target
python -m icakad shorturl update slug https://new-target
python -m icakad shorturl list --output links.json
python -m icakad shorturl delete slug --quiet

Paste Commands

python -m icakad paste create --text "hello"
python -m icakad paste create --text-file README.md --id doc
python -m icakad paste get doc --raw --output doc.txt
python -m icakad paste list --output pastes.json

Tips

  • Combine with shell pipes (... --raw > notes.txt) or set --output for repeatable sharing.
  • Use --quiet when scripting to suppress duplicates in stdout.
  • Running python -m icakad without arguments opens these docs automatically.
