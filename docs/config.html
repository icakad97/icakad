Config File

Place a JSON file at ~/.config/icakad/config.json or pass --config path.

{
  "shorturl_base": "https://linkove.icu",
  "paste_base": "https://linkove.icu",
  "token": "your-secret-token"
}

Any field can be omitted; missing values fall back to defaults.

Environment Variables

  • ICAKAD_CONFIG — custom config path.
  • ICAKAD_TOKEN — bearer token for both services.
  • ICAKAD_SHORTURL_BASE / ICAKAD_PASTE_BASE — override endpoints dynamically.

Programmatic Use

from icakad import load_settings, list_short_links

settings = load_settings(token="demo-token")
links = list_short_links(token=settings.token, base_url=settings.shorturl_base)

Combine settings with save_to to generate reports automatically:

list_short_links(save_to="links.json")
fetch_paste("release-notes", raw=True, save_to="notes.txt")
