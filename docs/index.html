icakad Toolkit

Unified helpers for icakad services running on xp97.icu, linkove.icu, and the @icakad GitHub projects. Build short links, share pastes, and automate workflows without digging through code.

Quick Start

Install the public package from PyPI and explore the live services.

python -m pip install icakad
python -m icakad --help

If you run python -m icakad without arguments the tool opens this documentation for you.

See available commands

What’s Included

  • Short URL client for the @shorturl worker on linkove.icu/api.
  • Pastebin client layered on top of the @paste worker.
  • Config loader that understands JSON files, environment variables, and CLI overrides.
  • File-friendly helpers so non-developers can write outputs to disk or read pastes from text files.

Repositories

All projects sync through GitHub:

  • @icakad — PyPI client (this repository).
  • @shorturl — Cloudflare worker powering short links.
  • @paste — Cloudflare worker for pastes and text shares.
