Create a Paste

from icakad import create_paste

create_paste(text="Hello xp97!", paste_id="demo-note", ttl=3600)

Arguments:

  • text or text_file — inline content or a file path.
  • paste_id — optional custom slug.
  • ttl — expire after N seconds (omit to keep forever).
  • as_plaintext — set to True to send text/plain instead of JSON.
  • save_to — store the JSON response so anyone can open it.

CLI Examples

# create from inline text
python -m icakad paste create --text "hello" --id demo

# create from a file and save the response
python -m icakad paste create --text-file notes.txt --output paste.json

# fetch raw text and pipe to a file
python -m icakad paste get demo --raw --output demo.txt

Sharing Workflow

  • Use fetch_paste(..., raw=True, save_to="note.txt") to hand off the plain text.
  • Combine with sys.argv in small scripts to generate release notes for @icakad and @paste simultaneously.
  • Remember that paste metadata is available through list_pastes() for dashboards or audits.
