Python Helpers

Use the high-level functions exported from icakad.

from icakad import add_short_link, list_short_links

add_short_link("promo", "https://icaka.eu/promo")
links = list_short_links()
print(links["promo"])

All helpers accept optional parameters:

  • base_url — point to staging (e.g. https://xp97.icu).
  • token — supply bearer tokens when the worker is locked down.
  • save_to — write responses to JSON (useful for sharing with non-technical teammates).

CLI Recipes

# create or update a slug
python -m icakad shorturl add my-link https://icaka.eu/

# list every slug and save to disk
python -m icakad shorturl list --output links.json

# delete a slug quietly
python -m icakad shorturl delete my-link --quiet

Combine with --config or --shorturl-base to target alternative environments.

Automation Tips

  • Run scripts from xp97.icu for lowest latency.
  • When working offline, capture list_short_links(save_to="links.json") and share the file.
  • Use branch names like codex/shorturl-refresh when committing updates that touch @shorturl.
